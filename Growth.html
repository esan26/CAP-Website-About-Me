Growth

How did I grow over the year?

It wasn't too hard for me to start out because I have a vivid memory of me using Code.org before for playing games so I recognized the UI.

I my first major project for the class was the Design an App Project where we learned how apps are designed and how they work. I made a app that was about finding art stores based on what art material the user was looking for. I spent a lot of time on the UI because each button, text, and icon needed a name for it which was a hassle for me to follow because I struggle to read properly.

From here I learned that I was a bit of a perfectionist when I worked on my projects, being that I spent so long on the littlest things. I did what was required on the rubric but I still spent so much time focusing on details when it was not needed.

On my next project I learned to simply "get over it" I can't think of a nicer work but I just taught myself that it's okay if stuff isn't pixel to pixel because what the main point of coding is that I understood how it works.
