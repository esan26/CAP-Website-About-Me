My Best Work

What was my best work for Comp Science?

My best work was working on the Game Project where we had to create a game. I created a game based off the Chiikawa Characters Usagi and Momonga

I learned that I didn't have to make everything perfect, buttt I wanted to have a lot of fun making this game so I looked for extra tips on the internet where I learned to make a "start" scene, make stuff that respawned, timed sequences, and added sound effects. It was very hard to add all of this together because it felt like one issue was happening after another. But I just had to learn to pirate do and I ended up having to not add a few sound effects because it would cause it to repeat over annoyingly so I dropped it because it didn't need to be perfect, I just had to learn and have fun making it.

I really loved this project, I got to learn a lot through it and be patient with it since coding took hours. This project lives as a reminder that there so many man-made resources on the internet, you just have to take the time to sit down and learn.
